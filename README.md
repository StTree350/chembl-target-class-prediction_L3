# Target Class Prediction for L3 ChEMBL Classes

This educational project was developed for the **Big Data and Machine Learning** course at **Saint Petersburg State University**.

The project includes four models based on different train-validation split strategies. These models are designed to classify the 11 most common L3 target classes in ChEMBL. They can also be used for class-like prediction of rare or previously unseen L3 classes by assigning them to the closest known class.

**Use of AI**
This project includes code that was partially generated uisng AI-assisted development tools. The final implementation was accepted by the author
